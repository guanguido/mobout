# MobOut – Angelgruppe Website

Statische Website für die Angelgruppe MobOut (www.mobout.de), Region Garmisch-Partenkirchen.

## Struktur

```
mobout/
├── index.html          # Haupt-HTML (Single-Page, enthält CSS + JS)
├── assets/
│   ├── images/         # Fotos (werden noch hinzugefügt)
│   └── data/           # Excel-Tabellen und sonstige Datendateien
└── CLAUDE.md
```

## Tech Stack

- Reines HTML/CSS/JavaScript (kein Framework, kein Build-System)
- Single-Page mit Smooth-Scroll-Navigation
- Responsive Design (mobile-first)
